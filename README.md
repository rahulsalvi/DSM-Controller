# Distributed Shared Memory Controller
Synchronizes shared memory buffers between multiple machines

## Requirements
 - Boost:
   - System
   - Thread
   - Interprocess
   - Asio
   - Program\_options

## Optional
 - Boost:
   - Log
   - Python

## Compatibility
 - Currently tested using:
   - OSX 10.11.5
   - Clang 3.8
   - Boost 1.59
   - Python 3.5.1
   - CMake 3.5.2
